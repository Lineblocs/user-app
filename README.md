## Lineblocs User Portal

## Requirements

```
NPM
Node 11
```

## Install

```
npm install
npm install -g install gulp@3.9.1
npm install -g install node-sass@4.13.0 --unsafe-perm
npm install -g install bower@1.8.8
bower install
```

## Run

```
gulp serve
```

To view the app please goto http://localhost:9000/
